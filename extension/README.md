# 📋 Clipboard Pro Manager

A modern Chrome Extension that automatically saves your copied text history and lets you reuse it anytime with a clean and fast UI.

---

## 🚀 Features

- 🧠 Automatically captures copied text
- 📜 Saves clipboard history locally
- 🔍 Real-time search functionality
- ⚡ One-click copy from history
- 💻 Supports code snippets & normal text
- 🎨 Modern dark UI design
- 🔒 Fully local (no server, private data)
